从数据分析到业务结论的闭环实践

两个项目分别聚焦医疗市场洞察与存储性能分析，均包含完整过程与结果。

项目 1：结直肠癌内窥镜市场偏好洞察

2025.02 - 2025.05

用户画像构建 + 聚类分群 + 推荐策略落地

从数据清洗到策略输出的一体化流程
  • 输入数据：整合 600+ 条采购数据与 300+ 份临床反馈。
  • 关键动作：SQL + Python 清洗数据并做特征工程，构建医疗机构用户画像。
  • 关键动作：采用 K-Means 聚类，将机构分为高精准度导向型与成本敏感型。
  • 关键动作：搭建规则 + 协同过滤混合推荐模型，利用决策树识别 Top3 关键因素（含采购预算）。

结果：聚类轮廓系数 0.82，Top-3 推荐命中率 78%。

项目 2：闪存仿真系统性能分析与可视化平台

2025.08 - 2025.12（独立开发）

性能数据建模 + 相关性分析 + Power BI 看板

围绕 IOPS、带宽、延迟做瓶颈识别
  • 输入数据：基于 MQSim 仿真日志，建立 SSD 性能数据集（IOPS、带宽、平均/尾延迟）。
  • 关键动作：SQL 进行表结构设计、多表关联与聚合；Python 完成清洗与宽表建模。
  • 关键动作：用分布分析 + 时间序列分析识别不同参数配置下的性能瓶颈。
  • 关键动作：用 NumPy/Matplotlib 计算 Pearson 相关系数并绘制热力图；Power BI 搭建联动看板。

结果：可快速定位异常波动与关键指标组合，提升参数调优效率。
